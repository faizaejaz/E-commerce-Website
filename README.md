# E-commerce-Website
Laravel E-Commerce Website with APIs

```
-> composer install

-> php artisan key:generate 

-> php artisan db:seed --class=ProductSeeder

-> php artisan passport:install

-> php artisan serve

```
```
php artisan tinker

User::factory()->count(10)->create()

Order::factory()->count(10)->create()

exit

```
